# Tactical Design

## Tactical Design

* 전략적 설계에서 더 상세한 부분 (Bounded Context 내부) 모델링
* Model Driven Design
* 계층형 아키텍처를 통한 도메인 모델 분리
* 도메인 이벤트를 통해 도메인을 보다 명확히 모델링
